# FAN Studio TileMap API

## 如何使用

请在您的地图应用中使用以下格式的 URL ：

`https://tilemap.fanstudio.tech/{source}/{z}/{y}/{x}`

其中 `{source}` 是下方列出的地图源标识之一。

## 可用的地图源

### ArcGIS 卫星影像

地图源标识 (source): `arcwi`

**URL:**

`https://tilemap.fanstudio.tech/arcwi/{z}/{y}/{x}`

[查看示例瓦片](https://tilemap.fanstudio.tech/arcwi/7/52/101)
